# Autofix Bot Claude Code Plugin

[Autofix Bot](https://autofix.bot) is the AI agent purpose-built for deep code review. Ship non-slop!

## Installation

In Claude Code, run:

```
/plugin marketplace add DeepSourceCorp/autofix-bot-claude-plugin
/plugin install autofix-bot@autofix-bot-claude-plugin
```

## Usage

Simply run:

```
/autofix
```
